## 学习目标

这份大纲的目标不是“会调库”，而是“先用 Python 手写，再理解框架为什么这么设计”。

学完后，你应该能做到：

1. 用 `Python + NumPy` 手写张量运算、损失函数、优化器和简单神经网络。
2. 理解前向传播、反向传播、梯度下降、正则化、初始化这些核心机制。
3. 从零实现线性回归、Softmax 回归、多层感知机、卷积层、简单 RNN。
4. 看懂 D2L 里 PyTorch 的简洁实现，并能把“手写版”和“框架版”对上。
5. 最后能独立完成一个小型深度学习项目。

## 学习原则

1. 每个主题都按“先原理，后代码；先手写，后框架”的顺序学。
2. 先做 `scratch` 版本，再看 `concise` 版本。
3. 每周至少留一个“纯手写作业”，尽量不用自动求导。
4. 数学只学够用的部分，以“能写出代码”为标准。
5. 学完一个模块就做一次小实验，不堆积。

## 推荐工具

- 必备：`Python 3.10+`、`NumPy`、`Matplotlib`、`Jupyter`
- 第二阶段后加入：`PyTorch`
- 可选：`pandas`、`tqdm`

## 建议学习顺序

先把前 8 周学完，再决定是否继续做 RNN 和 Attention。对于“入门 + 手写实现”来说，前 8 周是最关键的主干。

## 12 周教学大纲

### 第 1 周：准备环境与 Python 数值计算基础

- 对应 D2L：
  - `chapter_installation/index`
  - `chapter_preliminaries/ndarray.md`
  - `chapter_preliminaries/linear-algebra.md`
  - `chapter_preliminaries/calculus.md`
  - `chapter_preliminaries/probability.md`
- 目标：
  - 熟悉 `NumPy` 数组、广播、切片、矩阵乘法。
  - 知道梯度、偏导、链式法则、期望、方差的直觉。
- 手写任务：
  - 用 `NumPy` 实现向量加减乘除、矩阵乘法、均值方差。
  - 自己写一个小函数做数值微分，观察导数近似。
- 本周产出：
  - `numpy_basics.ipynb`
  - `numerical_grad.py`

### 第 2 周：数据、张量与自动求导的思想

- 对应 D2L：
  - `chapter_preliminaries/autograd.md`
  - `chapter_linear-networks/image-classification-dataset.md`
- 目标：
  - 理解“计算图”和“反向传播到底在算什么”。
  - 学会 mini-batch 数据读取。
- 手写任务：
  - 不依赖框架，写一个最简版 `DataLoader`。
  - 用“数值梯度”和“手推梯度”验证一元/二元函数。
- 本周产出：
  - `data_loader.py`
  - `grad_check.py`

### 第 3 周：线性回归从零实现

- 对应 D2L：
  - `chapter_linear-networks/linear-regression.md`
  - `chapter_linear-networks/linear-regression-scratch.md`
  - `chapter_optimization/gd.md`
  - `chapter_optimization/sgd.md`
- 目标：
  - 理解监督学习流程：数据、模型、损失、优化。
  - 能手写均方误差和随机梯度下降。
- 手写任务：
  - 用 `NumPy` 从零实现线性回归。
  - 训练并画出 loss 曲线。
- 本周产出：
  - `linear_regression_scratch.py`
  - 一份训练日志和损失图

### 第 4 周：Softmax 回归与分类问题

- 对应 D2L：
  - `chapter_linear-networks/softmax-regression.md`
  - `chapter_linear-networks/softmax-regression-scratch.md`
- 目标：
  - 理解分类、one-hot、Softmax、交叉熵。
  - 知道为什么分类损失常用交叉熵。
- 手写任务：
  - 用 `NumPy` 实现 Softmax 回归。
  - 在 Fashion-MNIST 或自选小数据集上做分类。
- 本周产出：
  - `softmax_regression_scratch.py`
  - `metrics.py`，至少包含 accuracy

### 第 5 周：多层感知机与反向传播

- 手写任务：
  - 用 `NumPy` 实现 `Linear -> ReLU -> Linear -> Softmax`。

### 第 6 周：训练技巧与稳定性

- 手写任务：
  - 给第 5 周的 MLP 加上 L2 正则。
  - 实现 Dropout 的训练/测试两种模式。
  - 比较不同初始化方式的收敛速度。

## 每周固定学习流程

1. 先读 D2L 对应章节，做一页手写笔记。
2. 用纸或平板把公式推一遍，尤其是梯度。
3. 先自己写，再看 D2L 的 `scratch` 代码对照。
4. 最后再看 `concise` 版本，理解框架帮你省掉了什么。
5. 每周末做一次复盘：这周我到底“手写了什么”。

## 学到什么程度算入门成功

满足下面 5 条，基本就算真正入门了：

1. 能不用框架写出前向传播。
2. 知道损失函数和梯度怎么来的。
3. 能解释为什么要用 mini-batch、初始化、正则化。
4. 能读懂 PyTorch 版 D2L 代码，而不是只会复制。
5. 能独立完成一个小项目并解释模型设计。
