# Generative Recommendation

This category collects papers about generative retrieval, Semantic IDs, LLM-based recommendation, and foundation-model style recommender systems.

1. [PLUM: Adapting Pre-trained Language Models for Industrial-scale Generative Recommendations](./plum-industrial-scale-generative-recommendations/README.md)

## Why This Category Matters

Generative recommendation changes the retrieval problem from vector matching to token generation. A good reading note in this category should explain:

1. What is being generated: item IDs, Semantic IDs, dense vectors, or natural language.
2. How items are tokenized and mapped back to real corpus objects.
3. How user behavior, item metadata, and multimodal content enter training.
4. How continued pre-training, SFT, decoding, and serving fit together.
5. What new system risks appear: invalid tokens, collisions, latency, candidate diversity, index refresh, and fallback.
