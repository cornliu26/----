# Final Project Template

Use this folder after lesson 12.

## Pick one direction

1. image classification with your own MLP or CNN
2. character-level text generation with RNN or GRU
3. a mini framework showcase with several handwritten modules

## Minimum project structure

```text
final_project/
├── README.md
├── report.md
├── experiments/
├── notebooks/
└── src/
```

## What to include in your report

1. task background
2. dataset description
3. model structure
4. what you implemented by hand
5. training setup
6. results
7. mistakes you hit and how you fixed them

## Suggested milestone plan

1. day 1: choose topic and prepare data
2. day 2: run a baseline
3. day 3: improve one component
4. day 4: write report and clean code

## A good final project is not necessarily huge

A small project is enough if you can clearly explain:

- why the model is built that way
- which parts you wrote yourself
- what the training curve means
- what failed and what you learned
